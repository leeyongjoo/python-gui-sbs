# python-gui-sbs
💰python으로 구현한 간단한 은행 시스템(Simple Banking System)

## Installation
[pip](https://pip.pypa.io/en/stable/)를 이용하여 라이브러리 설치 *Python Version: 3.8*
```bash
pip install -r requirements.txt
```

## Features
1. tkinter
2. socket
3. pycryptodome
4. threading
